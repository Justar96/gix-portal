# Product Overview

Gix is a self-hosted peer-to-peer file sharing desktop application with enterprise-grade security.

## Core Purpose
- Real-time file synchronization between peers without central servers
- End-to-end encrypted file sharing with granular access control
- Collaborative features: file locking, conflict resolution, presence awareness

## Key Features
- Create and manage shared drives (local folders synced P2P)
- Invite users via `gix://` deep links with permission levels (read/write/manage/admin)
- Real-time sync status, transfer progress, and activity feed
- File locking (advisory/exclusive) to prevent edit conflicts
- Automatic conflict detection with resolution strategies (keep local/remote/both/merge)
- Online presence tracking for collaborators

## Target Users
- Teams needing secure file sharing without cloud dependencies
- Users requiring self-hosted, privacy-focused collaboration
- Environments where data sovereignty is critical
